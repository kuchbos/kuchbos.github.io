some software packages I’ve written for the R programming language

replay of an AlphaGo training match using the kaya package

  • yamlform - a plaintext interview format for creating XLSForm documents for the ODK software platform
  • longformer - longitudinal database management in R
  • yamltools - a suite of tools for parsing YAML-structured plaintext interview output
  • ecodata - tools for managing transcription efforts in our research group
  • SIMani - agent-based simulation of multiple generations of demographically-realistic human populations
  • journaltools - a R package for keeping a simple plaintext journal
  • kaya - parsing and analyzing Go games stored in the .sgf format
  • chessparse - parsing and analyzing Chess games stored in the .png format
  • pacmanjr - lightweight package manager for an R codebase
